Game Check Point

This is the game check point which explains how my sudoku game is progressing

Logic

In my gameState i have one solutionGrid which is an array of the game solution. Then I have 4 others grid which given to the game dificulty level, easy, medium, hard and impossible. The game will give players some numbers in the grid and players then fill in the missing one. I put all the given numbers to negative and the players input will be zero before playing. That way I can distinguish which one is given and which one is input. I will update the level grid from player input and then compare the array with the solution array. There is a timer that will start when the game start. There also a Hint button for player to check how they are progressing. Besides, I will also check if player enter digit or not.
